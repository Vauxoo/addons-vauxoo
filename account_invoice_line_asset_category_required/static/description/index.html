Account Invoice Line Asset Category Required

This module set the policy for the asset category field in account invoice line. If you select:

  • 'Optional', the accountant is free to put a asset category on an account invoice line with this type of account.
  • 'Always', the accountant will get an error message if there is no asset category.
  • 'Never', the accountant will get an error message if a asset category is present.
