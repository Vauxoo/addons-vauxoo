Sale Block Cancellation

Added a new button in Sale Orders to only allow cancel a SO when all the pickings related with this are on ready to transfer or in waiting for another move state, or even with pickings on different states if the moved product quantities are correctly returned.

The original button to cancell is added to a new group Force Sale Cancel to can allow cancel a Sale Order out of these cases.

Installation

To install this module, you need to:

  • Not special pre-installation is required, just install as a regular Odoo module:
    • Download this module from Vauxoo/addons-vauxoo
    • Add the repository folder into your odoo addons-path.
    • Go to Settings > Module list search for the current name and click in Install button.
