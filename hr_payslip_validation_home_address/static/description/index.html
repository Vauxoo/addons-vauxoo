Hr Payroll Home Address Validation

This module adds validation for the field home_address of the employee in payroll Just have to install the module, not to do any configuration. This field home_address is found in form view HR / Payroll Employee / Employee_id in "Personal Information" tab If the field home_address is empty you can not confirm that employee's payroll and will see the message that the employee needs to have a home address, otherwise if may confirm the payroll.
