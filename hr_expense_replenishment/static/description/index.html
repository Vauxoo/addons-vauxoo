Expenses Replenishment

This module add the functionality to the HR Expense module to manage deductible expenses by using invoices asociated to an expense document. Also make an automation of the reconciliation process for the expense and the employee payment.

NOTE : You can download the account_invoice_line_currency module from the addons-vauxoo repository.
