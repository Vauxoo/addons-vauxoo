Stock Location Code

Add the code field to the location model. The location code is unique per company and will fo with the location name always. This way when searching a a location you can search it by name or code. The location record will be show "[code] name" location in the many2one fields (This emule the product model of product module).

New feature: Add the warehouse name to location. This way when searching a location by name or code if is it related to a warehouse the location record will be show "[code] name (warehouse)" in the many2one fields.
